Maze Solver

Category: Programming
Maze in blank

Example of the kind o mazes the program can solve. Note that each path must be 1px of width.

Example of the first maze solved by the program. The dark gray pixels are the beginning and final points, and the lighter gray is the path generated by the program.

Description

My team and I developed this program using a path searching algorithm as our final project for the Data Structures and Algorithms class. We programmed it in C++ and used Visual Studio as our IDE. I’ll leave the link to the project repository right here.
